## 📝 Как загрузить в Pinterest
1. Зайди в [Pinterest](https://www.pinterest.com)
2. Настройки → Импорт контента → Загрузить файл CSV
3. Выбери файл `pinterest_upload.csv`
4. Убедись, что картинки доступны по ссылкам
5. Выбери доску и опубликуй
