## 📝 Как загрузить в Pinterest
1. **Залей папку на GitHub**
   - Загрузи всю папку `for_pinterest` в репозиторий `pinterest-rss`
   - Убедись, что GitHub Pages включён

2. **Загрузи CSV в Pinterest**
   - Зайди в Pinterest Business → Настройки → Импорт контента
   - Нажми "Загрузить файл CSV"
   - Выбери файл `pinterest_upload.csv`
   - Проверь превью всех пинов
   - Выбери доску
   - Нажми "Опубликовать"
